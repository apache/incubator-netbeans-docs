Uncategorized changes

Adding ON_SAVE_REFORMAT

Aug 28 '12; API spec. version: 1.44; made by: mmetelka; issues: #140719

Adding new constants SimpleValueNames.ON_SAVE_REFORMAT and SimpleValueNames.ON_SAVE_USE_GLOBAL_SETTINGS.

Adding BRACE_SHOW_OUTLINE and BRACE_FIRST_TOOLTIP

Jul 5 '12; API spec. version: 1.43; made by: sdedic; issues: #214755

Adding new constants SimpleValueNames.BRACE_SHOW_OUTLINE and SimpleValueNames.BRACE_FIRST_TOOLTIP which control brace outline behaviour

Adding SimpleValueNames.EDITOR_SEARCH_TYPE

Apr 12 '12; API spec. version: 1.40; made by: mkristofic; issues: #210700

Adding new constant SimpleValueNames.EDITOR_SEARCH_TYPE.

Adding SimpleValueNames.ON_SAVE_REMOVE_TRAILING_WHITESPACE

Jun 9 '10; API spec. version: 1.33; made by: vstejskal; issues: #157561

Adding new constant SimpleValueNames.ON_SAVE_REMOVE_TRAILING_WHITESPACE.

Adding SimpleValueNames.NON_PRINTABLE_CHARACTERS_VISIBLE

Mar 4 '10; API spec. version: 1.30; made by: vstejskal; issues: #4619

Adding new constant SimpleValueNames.NON_PRINTABLE_CHARACTERS_VISIBLE.

Adding SimpleValueNames.TEXT_LINE_WRAP

Mar 4 '10; API spec. version: 1.29; made by: vstejskal; issues: #89894

Adding new constant SimpleValueNames.TEXT_LINE_WRAP.

Adding SimpleValueNames.ALT_HYPERLINK_ACTIVATION_MODIFIERS

Feb 16 '09; API spec. version: 1.24; made by: vv159170; issues: #157705

Adding a new constant SimpleValueNames.ALT_HYPERLINK_ACTIVATION_MODIFIERS as the name of an editor setting that determins shortcut for activating alternative (other than go to declaration) hyperlink action.

Adding SimpleValueNames.JAVADOC_POPUP_NEXT_TO_CC

Aug 28 '08; API spec. version: 1.22; made by: msauer

Adding new constant SimpleValueNames.JAVADOC_POPUP_NEXT_TO_CC.

Adding SimpleValueNames.THICK_CARET_WIDTH

Jun 3 '08; API spec. version: 1.21; made by: vstejskal

Adding new constant SimpleValueNames.THICK_CARET_WIDTH.

Added EditorStyleConstants.RenderingHints constant

May 27 '08; API spec. version: 1.20; made by: vstejskal

Adding new constant EditorStyleConstants.RenderingHints.

Added code folding related constants to SimpleValueNames

May 26 '08; API spec. version: 1.19; made by: vstejskal

Adding new constant to SimpleValueNames for code folding related settings.

Added switch for auto completing pair characters

May 13 '08; API spec. version: 1.18; made by: vstejskal; issues: #108161

Adding new constant to SimpleValueNames as the name of an editor setting that determins if pair characters should be autocompleted or not.

Added border related constants to EditorStyleConstants

Feb 28 '08; API spec. version: 1.16; made by: vstejskal; issues: #108161

Several new constants were added to EditorStyleConstants to support drawing rectangular boxes around text in the editor.

Added CodeTemplateDescription.getMimePath

Feb 14 '08; API spec. version: 1.15; made by: vstejskal; issues: #121663

The CodeTemplateDescription.getMimePath and a new constructor were added to accomodate a mime path where a code template was registered.

Added EditorStyleConstants.Tooltip

Sep 13 '07; API spec. version: 1.12; made by: vstejskal; issues: #107107

The EditorStyleConstants.Tooltip constant was added allowing to specify tooltips in highlighting layers.

Added CodeTemplateDescription.getUniqueId

Jun 15 '07; API spec. version: 1.11; made by: vstejskal

The CodeTemplateDescription.getUniqueId method was added and the constructor was extended to accept uniqueId parameter.

CodeTemplateSettings.getCodeTemplates siganture changed

Jun 10 '07; API spec. version: 1.10; made by: vstejskal

The CodeTemplateSettings.getCodeTemplates signature was changed to return Collection<CodeTemplateDescription> instead of List. Although an incompatible change this class had never been functional and should not have been used by anybody.

AttributesUtilities added

Jan 26 '07; API spec. version: 1.9; made by: vstejskal

The AttributesUtilities class has been added.

Deprecating FontColorSettings.PROP_FONT_COLORS

Jan 4 '07; API spec. version: 1.8; made by: vstejskal

Deprecating FontColorSettings.PROP_FONT_COLORS. It should have never been made public in the first place. Nobody can listen on this property anyway and the general contract is that settings instances in MimeLookup are immutable and if anything changes the whole instance (e.g. FontColorSettings) is replaced.

OpenIDE-Module-Needs: org.netbeans.api.editor.settings.implementation

Nov 13 '06; API spec. version: 1.7; made by: vstejskal

The module is now autoload and it needs the org.netbeans.api.editor.settings.implementation token. This token is provided by the editor/settings/storage module by default.

Private constructors in FontColorNames and SimpleValueNames

Nov 8 '06; API spec. version: 1.6; made by: vstejskal

The FontColorNames and SimpleValueNames classes used to have a default construtor, even though it had never made sense to create instances of them. The default constructors have been removed and a privat noargs constructor has been added to both of them preventing an accidental instantiation.

Editor Settings API created

Aug 26 '05; API spec. version: 1.2; made by: mroskanin; issues: #59784

Editor Settings API was created. It defines settings classes, which can be looked up via mimelookup.

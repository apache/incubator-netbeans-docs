    Details of all changes by API and date

    Common Server API

    The ServerInstance implements Lookup.Provider

    Jan 13 '12; API spec. version: 1.19; affected top-level classes: ServerInstance; made by: phejl; issues: #206196
    The ServerInstance became Lookup.Provider.

    Added UI helper method CommonServerUIs.showAddServerInstanceWizard

    Dec 22 '11; API spec. version: 1.18; made by: marfous; issues: #206479
    Added helper method invoking wizard for adding new server instance.

    Establish layer to register cloud providers

    Aug 24 '11; API spec. version: 1.15; made by: dkonecny
    Reuse existing server support provided by this module and use it to support concept of cloud providers. At the moment registration of cloud providers is implemented (/Cloud folder in layer). Their visualization will be added later. This is work in progress.

    Common Server Properties API

    Extensible actions on root server node

    Apr 3 '09; API spec. version: 1.4; made by: jtulach
    By registering actions to Servers/Actions you can extend the list of popup actions on Servers node in Services tab.

    Support API for server instance persistence

    Feb 10 '08; API spec. version: 1.1; made by: phejl
    Support API providing easy way for persistence of instance properties. Not manadatory for clients to use.

    Common Server SPI

    Initial version released

    Jan 12 '08; API spec. version: 1.0; made by: phejl
    First initial release of the Common Server SPI.
